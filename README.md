# 架构

- assets：用于存放静态资源文件，例如图片、字体等。
- components：存放公共组件。
- directives：存放自定义指令。
- filters：存放过滤器。
- mixins：存放混入对象。
- router：存放应用的路由信息配置。
- store：存放 Vuex 状态管理的相关文件。
- styles：存放样式文件。
- utils：存放通用工具函数。
- views：存放视图组件。
- App.vue：应用根组件。
- main.js：应用入口文件，主要进行 Vue 实例的初始化及引入全局依赖。
